# Vue-项目

## 项目技术栈

Vue3.2 + Vite4.1 + Pinia 2.0 + Element-Plus2.3 



## 推荐编辑器及插件

[VSCode](https://code.visualstudio.com/) + [Volar](https://marketplace.visualstudio.com/items?itemName=Vue.volar)  



## 项目运行

### 安装项目依赖

```sh
npm install
```

### 开发环境

```sh
npm run dev
```

### 编译打包

```sh
npm run build
```
